BIOS Recovery auf Supermicro-Boards

author: keanu | date: 2025-09-21 | tags: [hardware, supermicro]

Ein gebricktes Mainboard ist der Albtraum jedes Sysadmins. Supermicro hat Recovery-Mechanismen — aber die Doku hilft selten.

Voraussetzungen

USB-Stick, korrekte BIOS-Datei vom Support-Portal, und vor allem: nicht panisch werden.

// TIP: Immer Backup der aktuellen Config ziehen, auch wenn du denkst du brauchst es nicht.
# Boot von USB-Floppy emulieren via IPMI
$ ipmitool -I lanplus -H 192.168.1.100 \
    -U ADMIN -P password chassis bootdev floppy

# BIOS flashen
$ flashrom -p internal -w bios.rom

Der Rest ist Routine — wenn die Datei stimmt.
