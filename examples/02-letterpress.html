Cisco Catalyst Software Update Checklist

Keanu Fuchs · 21. November 2025

Before performing a software update on network devices, thorough and structured preparation is worthwhile. This checklist guides you through all relevant checks.

Check Operating Mode & Version

To check the currently running version, boot behavior, and uptime:

show version
show boot

Determine whether the system runs in Install Mode or Bundle Mode.

Every problem solved is a lesson learned — and every lesson learned should be shared.

Backup Configuration

Save the running config. External backup via TFTP, FTP, or SCP empfohlen.
